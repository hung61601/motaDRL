# Mota DRL

Mota DRL 是「使用深度強化學習的魔塔 AI」(Mota AI using Deep Reinforcement Learning) 的簡寫。

本專案實現了使用 Graph Isomorphism Network 結合 Proximal Policy Optimization 來學習遊戲策略。

- 作者：Hung1

## 相依套件 Dependencies
- Python 版本為 3.11.5
- 安裝相依套件可在根目錄下使用指令：
```bash
pip install -r requirements.txt
```
